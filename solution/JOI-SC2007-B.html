# JOI 春合宿 2007 day1-2 Factorial

# 問題

  • https://atcoder.jp/contests/joisc2007/tasks/joisc2007_factor (opens new window)

# 解説

n を以下のように素因数分解したときのことを考える.

n=p1e1⋅p2e2⋅p3e3⋅⋅⋅pkek n = p_{1}^{e_1} \cdot p_{2}^{e_2} \cdot p_{3}^{e_3} \cdot \cdot \cdot p_{k}^{e_k} n=p1e1​​⋅p2e2​​⋅p3e3​​⋅⋅⋅pkek​​

このとき, もし指数 eie_iei​ が全て 1 ならば,

n=p1⋅p2⋅p3⋅⋅⋅pk n = p_{1} \cdot p_{2} \cdot p_{3} \cdot \cdot \cdot p_{k} n=p1​⋅p2​⋅p3​⋅⋅⋅pk​

となり, m!m!m! が nnn で割り切れる最小の mmm は素因数 pip_ipi​ の最大値となる.

n=12=22⋅3n = 12 = 2^2 \cdot 3n=12=22⋅3 のときのように 指数 eie_iei​ に1ではないものがある場合もある. 求める mmm の必要条件は, m!m!m! が pieip_{i}^{e_i}piei​​ で割り切れることである. これを満たす最小の mmm は次のように求めることができる. m=pi,2pi,3pi,...kpim = p_i, 2p_i, 3p_i, ... kp_im=pi​,2pi​,3pi​,...kpi​ のように pip_ipi​ の整数倍を順に試していき, m!m!m! の ppp の素因数の指数を数えて初めて eie_iei​ 以上となった mmm を採用する.

この操作を全ての素因数に対して行い, それらの最大値を取れば求める答えが得られる.

# 実装例

# C++

  • submission: https://atcoder.jp/contests/joisc2007/submissions/26205383 (opens new window)
#include <algorithm>
#include <iostream>
#include <map>

std::map<long long, int> prime_factor(long long n) {
  std::map<long long, int> res;
  for (long long x = 2; x * x <= n; x++) {
    while (n % x == 0) {
      ++res[x];
      n /= x;
    }
  }
  if (n != 1) {
    res[n] = 1;
  }
  return res;
}

int main() {
  int n;
  std::cin >> n;

  std::map<long long, int> prime_factors = prime_factor(n);

  long long ans = 0;
  for (auto it = prime_factors.begin(); it != prime_factors.end(); it++) {
    long long prime = it->first;
    int e = it->second;

    int cnt = 0;
    for (int i = 1; i <= n; i++) {
      long long m = prime * i;
      ans = std::max(ans, m);

      while (m % prime == 0) {
        cnt++;
        m /= prime;
      }

      if (cnt >= e) {
        break;
      }
    }
  }

  std::cout << ans << "\n";

  return 0;
}

# Python

  • submission: https://atcoder.jp/contests/joisc2007/submissions/26205492 (opens new window)
#!/usr/bin/env python3


def prime_factor(n):
    res, x = {}, 2
    while x * x <= n:
        while n % x == 0:
            res[x] = res.get(x, 0) + 1
            n //= x
        x += 1
    if n != 1:
        res[n] = 1

    return res


n = int(input())
prime_factors = prime_factor(n)

ans = 0
for prime, e in prime_factors.items():
    cnt = 0
    x = 1
    while cnt < e:
        m = prime * x
        x += 1
        ans = max(ans, m)

        while m % prime == 0:
            cnt += 1
            m //= prime

print(ans)
